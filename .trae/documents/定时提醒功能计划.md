# 定时提醒功能计划

## 功能概述
在待办事项的右键菜单中增加"定时提醒"入口，支持两种时间选择模式：
1. **选择时间点** — 使用 TimePicker 选择一天内的任意时间（HH:mm），当天该时间触发提醒，**已过的时间自动禁用**
2. **N 小时后提醒** — 输入数字，选择几小时后触发提醒（最大 12 小时）

到达指定时间后，通过操作系统原生通知进行提醒，通知内容为待办事项的内容。

**额外视觉需求**：设置提醒后，在待办项右侧显示时钟图标 + 实时倒计时（精确到分钟，显示"还剩 X小时Y分钟"），每分钟更新一次。

---

## 实现步骤

### Step 1: 安装 Tauri Notification 插件

**Rust 端** — 修改 `src-tauri/Cargo.toml`，添加依赖：
- `tauri-plugin-notification = "2"`

**前端** — 安装 npm 包：
- `@tauri-apps/plugin-notification`

**权限配置** — 修改 `src-tauri/capabilities/default.json`，添加 `notification:default` 权限

### Step 2: 注册 Tauri 插件

修改 `src-tauri/src/lib.rs`（或 main.rs），注册 notification 插件：
```rust
tauri::Builder::default()
    .plugin(tauri_plugin_notification::init())
    // ...
```

### Step 3: 扩展 TodoItem 类型

修改 `src/types/index.ts`，在 `TodoItem` 接口中新增字段：
```typescript
remindAt: string | null   // ISO 格式的提醒时间，null 表示无提醒
```

同时在 `TodoItemNode` 中加入同名可选字段。

### Step 4: 创建提醒调度服务 `src/services/reminderService.ts`

新建文件，职责：
1. 维护一个定时器映射表
2. **`scheduleReminder(todoId, remindAtISO, content, callback)`** — 注册定时器，到时执行回调
3. **`cancelReminder(todoId)`** — 取消指定待办的定时器
4. **`loadReminders(todos, callback)`** — 应用启动时扫描所有未过期的 remindAt，重新注册定时器
5. **`clearAllReminders()`** — 清理所有定时器

回调函数由 App.vue 传入，负责调用通知 API。

### Step 5: 创建 ReminderDialog 组件

新建 `src/components/ReminderDialog.vue`：
- Props: `open`, `todoId`, `todoContent`, `currentRemindAt`
- 两种模式切换（`a-radio-group`）：
  - **指定时刻**：`a-time-picker` format="HH:mm", minute-step=5
    - 使用 `disabledTime` prop 动态禁用当前时间之前的时/分
  - **N 小时后**：`a-input-number` (min=1, max=12) + "小时" 标签
- 确认时 emit `confirm(remindAtISO)`，取消时 emit `cancel`
- 如果已有提醒，显示"取消提醒"按钮

### Step 6: 修改 TodoItem.vue

1. **右键菜单**：增加"定时提醒"菜单项，使用 `ClockCircleOutlined` 图标，key: `reminder`
2. **倒计时显示**：在模板中，当 `todo.remindAt` 存在且未过期时，在右侧渲染：
   - 时钟图标 + 倒计时文本（如"还剩 2小时30分钟"）
   - 使用 `setInterval` 每 **60 秒**更新一次
   - 倒计时归零时隐藏
3. emit: `set-reminder`, `cancel-reminder`

### Step 7: 事件冒泡

在 `NestedTodoList.vue` 和 `TodoList.vue` 中透传新的事件。

### Step 8: 修改 App.vue

1. 管理 `reminderTodoId` ref 控制 ReminderDialog 显示
2. 处理提醒相关事件
3. onMounted 调用 `loadReminders` 恢复提醒定时器
4. 删除/完成待办时取消提醒
5. 通知回调：Tauri 环境用 `sendNotification`，浏览器用 `Notification API`

### Step 9: 倒计时更新机制

倒计时在 TodoItem 中实现：
- 通过 `computed` 根据 `todo.remindAt` 实时计算剩余分钟数
- 使用 `setInterval` 每 60 秒触发一次重新计算（trigger computed 重新求值）
- 剩余时间显示格式：
  - 剩余 >= 1 小时：显示"还剩 X小时Y分钟"
  - 剩余 < 1 小时：显示"还剩 X分钟"
  - 剩余 <= 0：停止 interval 并隐藏倒计时 UI

## 组件树 & 事件流

```
App.vue
 ├─ TodoList
 │   └─ NestedTodoList
 │       └─ TodoItem  (右键菜单 → "定时提醒" + 倒计时显示)
 └─ ReminderDialog     (配置弹窗)
```

## 关键注意事项

1. **Tauri 通知权限**：需注册插件并配置 capability
2. **时间禁用**：TimePicker 使用 `disabledTime` 属性动态计算
3. **定时器持久化**：`remindAt` 存 SQLite，重启后通过 loadReminders 恢复
4. **已完成/已删除**：自动取消提醒
5. **浏览器降级**：非 Tauri 用 `Notification API`
6. **倒计时更新频率**：60 秒 interval，而非 1 秒
